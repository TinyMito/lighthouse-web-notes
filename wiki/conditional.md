# Conditional

### if...else
Check condition
```javascript
if (a === b) {
  // do stuff
} else {
  // do stuff
}
```

Check if truthy
```javascript
if (a) {
  // do stuff if true
} else {
  // do stuff if false
}
```

Check if falsey
```javascript
if (!a) {
  // do stuff if false
} else {
  // do stuff if true
}
```

### ES6 Shorthand
```javascript
let result = a === b ? 'do true stuff' : 'do false stuff';
```
